# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In summary, we are committed to providing a welcoming and inclusive
environment for all contributors, regardless of background or
experience level. We expect all participants — maintainers,
contributors, reviewers, and users — to behave respectfully and in
good faith.

## Reporting

To report a violation, email **conduct@amp-protocol.dev**.

Reports are reviewed by the project maintainers. We follow the
enforcement guidelines defined in the Contributor Covenant 2.1
(Correction → Warning → Temporary Ban → Permanent Ban).

## Scope

This Code of Conduct applies in all project-controlled spaces:
GitHub issues and pull requests, the project's communication
channels, and any in-person or online events held under the project's
banner. It also applies when an individual is officially representing
the project in public spaces.
